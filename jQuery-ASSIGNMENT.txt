1.What is jQuery?
Ans. jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

2. How to Apply CSS Using JQuery,How to Add Class ans Remove Class in Jquery,JQuery Animation?
Ans. jQuery Manipulating CSS
       1. addClass() - Adds one or more classes to the selected elements.
       2. removeClass() - Removes one or more classes from the selected elements.
       3. toggleClass() - Toggles between adding/removing classes from the selected elements.
       4. css() - Sets or returns the style attribute.

    The jQuery animate() function performs custom animations on a set of CSS properties. Syntax: (selector).animate(properties [,duration][,easing][,complete]) Parameters: properties (*required): These define the objects of CSS properties and values to animate.
